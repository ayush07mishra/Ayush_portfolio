# Portfolio Website

This is my personal portfolio website built to showcase my projects, skills, and experience. It highlights selected work across development and problem solving, along with links to live demos and GitHub repositories.

The goal of this portfolio is simple: present my work in a clean, structured way and make it easy for recruiters or collaborators to understand what I’ve built and how I think.

## Features

- Responsive design for desktop and mobile  
- Project showcase section with preview and detail links  
- Clean UI with modern layout  
- Organized code structure for easy updates
